# Portfolio Website

## Project Description
This is my personal **Portfolio Website** built to showcase my projects, skills, and experience.  
It is designed using **Bootstrap** for a responsive and modern look.  

The portfolio includes sections like:
- About Me
- Projects (including Internship Management System)
- Contact
## Features
- Responsive design that works on desktop, tablet, and mobile.
- Showcase projects with images, descriptions, and links to GitHub.
- Clean and professional layout using Bootstrap.
- Easy to update and maintain.
## Technology Used
- **Front-end:** HTML, CSS, JavaScript, Bootstrap
- **Version control:** Git & GitHub
